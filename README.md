# Blinkit Sales Analytics Dashboard 📊🛒

An interactive, premium-designed Power BI dashboard developed to analyze Blinkit's (Quick-Commerce) sales performance, customer ratings, and outlet metrics across different regions, item types, and establishment timelines.

## 🎯 Project Objective
The goal of this project is to perform a comprehensive data analysis on grocery sales and derive actionable business insights. It tracks key performance indicators (KPIs) and uncovers hidden trends related to item visibility, fat content, outlet sizes, and location tiers.

## 🛠️ Tools & Tech Stack
- **Business Intelligence:** Power BI Desktop
- **Data Governance & Cleaning:** Power Query
- **Data Modeling:** Star Schema (Fact and Dimension concepts)
- **Analytical Expressions:** DAX (Data Analysis Expressions) for custom metrics
- **Design Elements:** Premium custom dark UI layouts with specialized asset icons

## 📊 Key Metrics & KPIs Tracked
- **Total Sales:** Overall revenue generated across all outlets.
- **Average Sales:** Mean sales value per transaction.
- **Number of Items:** Total count of unique products sold.
- **Average Rating:** Customer satisfaction metric across different segments.

## 📈 Executive Insights Discovered
- **Fat Content Analysis:** Low-fat items significantly outperform regular-fat items in total sales volume across almost all outlet categories.
- **Outlet Location Impact:** Tier 3 location types yield the maximum revenue stream, indicating higher demand or bigger ticket sizes in suburban/large zones.
- **Establishment Performance:** Outlets established around specific timelines show distinct operational metrics compared to older setups.
- **Item Type Distribution:** Fruits, Vegetables, and Snack Foods occupy the highest market share in order frequency and total revenue generation.
